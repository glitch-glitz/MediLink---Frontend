# MediLink Medical Supplies

MediLink Medical Supplies is a modern medical e-commerce platform built to help hospitals, clinics, pharmacies, laboratories and healthcare professionals browse and request quality medical equipment and healthcare supplies online.

The project focuses on delivering a clean, responsive, and user-friendly shopping experience while providing an efficient catalogue of medical products.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/medilink-medical-supplies.git
```

Navigate into the project

```bash
cd medilink-medical-supplies
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
